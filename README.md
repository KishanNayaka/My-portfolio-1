Premium 3D Interactive Web Developer Portfolio

Welcome to my personal portfolio repository! This is a high-performance, responsive single-page web application engineered to showcase my full-stack and front-end development capabilities. It features real-time 3D parallax effects, fluid interactive background physics, and seamless third-party backend integrations.

Live Demo: https://kishannayaka.github.io/My-portfolio-1/

---

Core Features & "Spicy" UI Mechanics

Native 3D Tilting Matrix:** Custom JavaScript-driven parallax mechanics embedded into the featured project cards, creating structural depth changes (`translateZ`) based on cursor coordinates or mobile touch inputs.
Reactive Particle Mesh Canvas:** An HTML5 `<canvas>` rendering engine fueling a fluid, floating lattice network that dynamically shifts away from user interaction vectors.
Floating 3D Geometric Objects:** CSS keyframe-animated floating shapes drifting across layered layout wrappers to maximize visual immersion.
Connected Form Pipeline:** Hardwired seamlessly to Formspree backend architectures, routing user inquiries straight to my professional inbox without requiring heavy external server platforms.
Interactive Scheduling Widget:** High-end client UI element rendering calendar metrics directly in the layout structure.
